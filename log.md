# 1일차-아이디어 구상
### 1일차에는 영현이와 함께 화아트보드로 뽀로로도 그려보고 여드름짜기나 축구게임도 만들어보자고 이야기를 했지만 포기하였다.
# 2일차-게임 구상
### 2일차에는 내가 고민을 하다 내가 커비를 하자고 해서 알겠다하고 커비를 떨궈서 득점하는 게임을 만들자라고 이야기 했다.
# 3일차-코딩시작
### 3일차에는 게임을 만드는 것은 무리라고 생각을 하여서 그냥 커비만 그리는걸로 하여서 커비 코딩을 시작했다.
#### 만든 코딩 *box(pos=vec(0, 0, 0),size=vec(400,400,400), color=vec(247/255,218/255,234/255))* 커비 몸통과 몸통에 색깔을 추가 하였다
# 4일차-코딩 재개
### 4일차에는 커비에 눈에 그라데이션도 넣고 눈동자와 입을 만들고있는중이었다.
#### 코딩 그라이데이션은 수업시간에 했던 내용바탕으로 하고 눈동자는 박스 코딩하여 넣었다 입은 안으로 좀 들어가는 모양을 만들생각이다
